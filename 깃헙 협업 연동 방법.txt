깃헙 협업 연동 방법
콜라보레이터 등록
등록된 사람은 메일을 통해 초대에 응해야 함
깃 설치
DOWNLoad for mac => install homebrew -> install 명령어 복사 -> 터미널 실행 -> 복붙 ->패스워드 요구 시 컴퓨터 비밀번호 치면 됨
Next steps : Run brew help to get started 뜨면 끝 
확인 차 명령어 : brew --verson -> 이러면 깃 버전이 떠야 함
이제 hombrew 깃 설치해야함 
brew install git
확인 차 명령어 : git -version -> 깃 버전이 떠야 함 

깃 개인 설정지정(git bash 혹은 터미널)
git config --global user.name "내이름"
git confit --global user.email "깃헙 가입 이메일"

이후 확인하기 위해서는
git config --list -> 실행 시 내가 입력한 이름과 메일이 하단에 뜨게됨


리포지토리(로컬)-깃헙 연결

cd (깃 리포지토리 위치로 이동)
git init (깃 초기화)
git add . (리포지도리에 있는 모든 파일을 깃에 올리겠다)
git status (새로 더한 파일이 리스트업된걸 보여줌/ 필수는 아님)
git remote add origin git@github.com:계정/리포지토리이름.git 
git remote -v로 나의 리포지토리와 깃헙이 연결되었는지 확인

깃헙과 리포지토리 연결에는 https나 ssh키 생성해서 연결하는 방식이 있는데
https 는 내 계정과 토큰(개인 계정에서 생성)을 매번 입력해야함
ssh는 내 컴퓨터와 깃헙 간의 인증 통로를 만들어서 한번 입력하면 자유롭게 푸쉬할 수있음
ssh로 바꾸는건 클로드와 상의해서 해결하면 된다
ssh 키 설정 
1. 키 있는지 확인 
ls -al ~/.ssh
no such file or directory나 아무것도 없으면 키가 없는것

2. 키 생성
ssh-keygen -t ed25519 -C "깃헙 이메일"
세번 물어보는데 전부 그냥 엔터

3. 공개 키 복사
pbcopy < ~/.ssh/id_ed25519.pub
(화면에 아무것도 안뜨지만 복사된것)

4. 깃헙에 SSh 키 등록

5. REMOTE 주소를 SSH 로 변경
git remote set-url origin git@github.com:아이디/리포지토리.git

체크하려면
git remote -v
이 때 git@github.com... 으로 바뀌면 된거임

6. 연결 테스트
ssh -T git@github.com
처음이면 연결 계속할거냐고 물어보는데 그때 yes 입력
hi 아이디! 이런 식으로 뜨면 성공

7. 푸시 
git push -u origin main(브랜치 이름마다 달라질 수있음)
