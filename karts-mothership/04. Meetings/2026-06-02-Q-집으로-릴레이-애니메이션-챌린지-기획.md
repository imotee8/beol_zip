# 집으로 릴레이 애니메이션 — 챌린지 카드 기획서

> **완성 비전**: 훅이 3번 반복되는데 — 볼 때마다 같은 걸음인데 왜인지 다른 사람 같다. 보는 사람이 어디에 서 있느냐에 따라 감정이 달라지는 애니메이션.

## 기획 핵심 구조 (확정)

| 항목 | 내용 |
|------|------|
| 음악 | 발밍 타이거 "집으로" — 반복되는 훅 3소절 |
| 캐릭터 | Mixamo **sad walk** (힘없이 걸어가는 모션) — 전원 동일 |
| 차이 | 카메라 구도/거리만 다름 (+ 후보정 에펙 카메라 이동) |
| 도구 | Blender + Mixamo → ComfyUI AI 렌더 → After Effects |
| 제약 | 3D에서 카메라 애니메이팅 **금지** (에펙에서 처리) |

**왜 이 구조가 강한가**: 같은 모션 + 다른 시점 = [[설명하지 않는 연출 — 의도적 모호성]]의 릴레이 버전. 관객이 "이게 한 장면인가? 다른 장면인가?" 사이에서 흔들림.

---

**카드 1 — 짭응이**

```
👁 트리거 — 이미지 2장 + 가사:

Step 1. Eagle / Pinterest / Cosmos 열기
        → 직관으로 마음에 걸리는 이미지 2장 뽑기
        → 기준 없어도 됨. 그냥 지금 끌리는 것.

Step 2. 발밍 타이거 "집으로" 가사 펼치기
        → 두 이미지 옆에 놓고 같이 본다

Step 3. 이미지 2장 + 가사를 보며 → 시 3줄 쓰기
        → 설명 아님. 감각만. 단어가 이어지지 않아도 됨.
        → 이 3줄이 내 씬의 방향이 된다.

💥 챌린지 #12 변형 (이미지 2장 + 가사 → 시 → 컷):
그 3줄에서 가장 강한 한 줄 → 마지막 프레임 1컷 스케치
Extreme Full shot — 배경이 캐릭터보다 큰 샷
애니메이팅은 의도적 저프레임 (로토스코핑 느낌)

⚡ 이미지 뽑는 데 10분 + 시 쓰는 데 10분 + 스케치 10분 = 30분
```

## ComfyUI 파이프라인 (첫 실습)

[[ComfyUI AI 렌더 워크플로우 가이드]] 기반:

```
Blender sad walk 렌더 (Outline Pass 추출)
    ↓
각 감독이 레퍼런스 이미지 1장 지정 (스타일 결정)
    ↓
ComfyUI AI Renderer 실행
    ↓
After Effects (카메라 이동 + 편집)
    ↓
음악 싱크 (발밍 타이거 "집으로" 훅)
```

**각 감독의 레퍼런스 이미지 = 이 작업의 핵심 창작 선택지.**
[[아트 스타일 조합 설계]] 관점에서: 레퍼런스 이미지 1장이 스타일 전체를 결정함.
